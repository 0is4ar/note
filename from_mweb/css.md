# css

```css
<style>
.blacklink{
    font-size:2.5em;
    color:black;
    text-decoration: underline;
    font-weight:bold;
}
</style>
```

- .blacklink代表修饰class='blacklink'的元素

- \#blacklink代表修饰id='blacklink'的元素

如果没有前缀直接

```css
p { font-size:2.5em;
    color:black;
    text-decoration: underline;
    font-weight:bold;
}
```
代表修饰所有的p元素
